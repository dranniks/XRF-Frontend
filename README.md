## Предметная область
Проект отображает каталог эталонных сплавов для рентгенофлуоресцентного анализа, позволяет искать услуги по названию, просматривать карточку одной услуги и работать с заявками на определение состава сплава.

В системе используются данные XRF-анализа: описание исследуемого артефакта, измеренные значения по элементам `Cu`, `Zn`, `Sn`, `Pb`, комментарии к позициям заявки и рассчитанный результат сопоставления с эталонным сплавом.
